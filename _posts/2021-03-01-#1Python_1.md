## 데이터 입력, 출력하기

### 명령어의 형식

```
1. print() ⟹ "출력해라" 라는 명령
2. input() ⟹ "입력과 관련된 일을 컴퓨터가 하게 하는 명령"
3. list() ⟹ "리스트를 작성하라"
4. next() ⟹ "다음으로 가라" 명령

명령어 형식 : 영어단어()
명령어 의미 : 컴퓨터야, 미리 정해진 명령 내용대로 일을 해라
```

## 출력

* 출력 명령어 print()

```python
print(123) # 123
print(2020) # 2020
print(가나다) # error

마지막은 에러가 발생한다 문자를 출력하기 위해서는 '', "" 을 사용해야 한다.

print("가나다") # 가나다


print(123) # 123
print('123') # 123
   # 두개는 다른 의미이다.
   # 첫 번째는 숫자, 두번째는 문자이다.

print(123 + 2) # 125
print('123' + 2) # error
   # 첫 번째는는 숫자이다
   # 두 번쨰는 에러가 발생한다.
   # 사람도 문자와 숫자를 더할 수 없으니 컴퓨터도 알아 듣지 못하기 때문이라 생각하면 된다.


# 사칙 연산
print(123 + 9) # 132
print(123 - 9) # 114
print('123 + 9') # 123 + 9
print('123 - 9') # 123 - 9


print(108 * 8) # 864
print(108 / 8) # 13.5

print(108 // 8) # 13
print(108 % 8) # 4
print(108 ** 8) # 18509302102818816
print(5 ** 2) # 25

# 여러 연결 요소 출력하기
print(14 + 31) # 45
print('파이썬' + '프로그래밍') # 파이썬프로그래밍
print('파이썬', '프로그래밍') # 파이썬 프로그래밍
print(14, 31) # 14 31
   # +(플러스) 말고도 ,(콤마)로 사용이 가능하며, ,로 사용할 시 공백이 생긴다.

# 관련 이건 무엇일까?
print('파이썬' + 3)
print('파이썬' - 3)
print('파이썬' / 3)
print('파이썬' * 3)
   # 에러인 것은 당연히 이해를 했을 것이다.
   # 하지만 *일 때는 예외이다.

print('파이썬' * 3) # 파이썬파이썬파이썬
   # 반복을 의미하게 된다.
```
